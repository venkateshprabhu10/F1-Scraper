# Formula 1 Scrapper

This is a Python based scrapper which scrapes details of the events like event location, start date and end date.

## Installation

Create a new virtual environment and use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements.

```bash
python -m venv venv
```

```bash
pip install -r requirements.txt
```

## Usage

Run the following command to get the list of events for the year 2023.
```bash
python .\f1-scrapper\f1-calendar-scrapper.py
```
